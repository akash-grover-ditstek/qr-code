    ## About Project
    
    **
    
    The project use to create simple QR Code image 
    
    clone project by 
    ```
    git clone https://github.com/akash-grover-ditstek/qr-code qr-code
    cd qr-code
    composer install
    php artisan storage:link
    php artisan serve
    goto http://localhost:8000
    ```
